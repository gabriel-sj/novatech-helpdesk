# NovaTech - Sistema de Suporte ao Usuário 🚀

Este projeto é um sistema de helpdesk funcional desenvolvido para a disciplina de Informática Educativa, focado em gerir chamados de suporte técnico da empresa **NovaTech**.

## 🛠️ Tecnologias Utilizadas
- **Backend:** Node.js com Express
- **Base de Dados:** SQLite3 (Relacional)
- **Frontend:** HTML, CSS e EJS (Templates dinâmicos)
- **Controle de Versão:** Git e GitHub

## 🚀 Como Instalar e Rodar
1. Clone o repositório:
   `git clone https://github.com/teu-usuario/novatech-helpdesk.git`
2. Entre na pasta:
   `cd novatech-helpdesk`
3. Instale as dependências:
   `npm install`
4. Inicie o servidor:
   `node server.js`
5. Aceda no navegador: `http://localhost:3000`
